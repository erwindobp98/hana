```shell
git clone https://github.com/erwindobp98/hana.git
```

cd hana

```shell
nano .env
```

```shell
PRIVATE_KEY=your_private_key
WALLET_ADDRESS=your_wallet_address
```
Install
```shell
pip install web3
```
Run
```shell
python bot.py
